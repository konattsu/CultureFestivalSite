80年以上未解決のコラッツ予想

コラッツ予想について

コラッツ予想とは「すべての正の整数において、奇数なら3倍し1を足す、偶数なら2で割る、 という操作を繰り返していくとどんな整数でも有限回数で1に到達する」という予想です。 また角谷の問題、ウラムの予想とも言われています。

具体例

一例として3を例にとります。3は奇数なので3倍して1を足すと10になります。10は偶数で2で割って5になります。5は奇数なので3倍して1を足し16になります。16は偶数で2で次々と4回割れるので1になります。この操作は以下のようになります。
3, 10, 5, 16, 8, 4, 2, 1

証明の進捗

現在までに2の68乗までの値が1に到達することが確かめられています。つまり、2の68乗以下の数字では1以外はどの数字も同じ数字に戻ることはなく(ループせず)、また限りなく大きくなっていくこともない、ということです。

ちなみに2021年には、日本企業の音圧爆上げくんが1億2000万円の懸賞金をかけました。
皆様もぜひコラッツ予想について調べてみてください。

調和数列と音楽の関係性

私はゲームなどのBGMをよく聞き、特にベースのコード進行など音階の構成を楽しんでいる。いきなり数学と全く関係のない話と思うかもしれないが、この音楽ととても深い関係のある数学の要素がある。それは数列である。
一体音楽と数列の間にどのような関係があるのか、調べて以下にまとめてみた。

1. 調和数列

本題に入る前に調和数列(等差数列の逆数の数列)についての例を、次に示す。

調和数列
数列{a[n]} = 20, 15, 12, 10,... (a[n] ≠ 0) が調和数列のとき
その逆数の数列{1/a[n]} = 1/20, 1/15, 1/12, 1/10,... は等差数列

よって一般項 1/a[n] = 1/20 + (n-1)/60 = (n-2)/60
したがって a[n] = 60/(n+2)

それでは本題に入りたいと思う。

2. 数列と音楽の関係性

1. 周波数と等比数列

音は空気の振動であり、周波数(1秒間の空気の振動数: Hz)で表すことができる。実はドレミの音階の周波数には等比数列が隠れている。
次にラの音から始まる1オクターヴ分の周波数を示す。

音階 周波数[Hz]
ラ 220
ラ# 233
シ 247
ド 262
ド# 277
レ 293
レ# 311
ミ 330
ﾌｧ 349
ﾌｧ# 370
ソ 392
ソ# 415
ラ 440

この表で下のラ(220Hz)を初項、公比: 233/220 ≒ 1.06, 項数12個として 周波数は

a[n] = 220・(1.06)^(n-1) の等差数列を成す。

実際、上のラの音は220・(1.06)^12 ≒ 442.6 となり、実際に数列となっていることがわかる。

2. 和音と等差、調和数列

音楽の構成要素として、和音(コード)がある。その中でも代表的な C(ドミソ), F(ﾌｧラド), G (ソシレ) の3つのコードについて見てみると、

コード 構成音 周波数比 弦の長さの比
C ドミソ 24, 30, 36 1/24, 1/30, 1/36
F ﾌｧラド 32, 40, 48 1/32, 1/40, 1/48
G ソシレ 36, 45, 54 1/36, 1/45, 1/54

周波数比: 上表,周波数÷11

上の表から、周波数比はそれぞれ
C[n] = 6n + 18,
F[n] = 8n + 24,
G[n] = 9n + 27
 の等差数列と表すことができ、弦の長さの比はそれぞれの逆数なのだから、調和数列といえる。実際、この弦の長さの比で作られる音が「調和」してコードを構成するから調和数列と呼ばれている。

3. 不協和音と数列

コードの中にはいわゆる不協和音と呼ばれるものもある。そういった物の場合はどうなるであろうか。

1. C7(ド,ミ,ソ,シ♭)
　周波数比{C7[n]} = 24,30,36,42 となり、ソとシ♭の間で規則性がなくなる。

2. Dm9(レ,ﾌｧ,ラ,ド,ミ 但しレ,ﾌｧはラより低いものとする)
　周波数比{Dm9[n]} = 13,16,20,24,30 となり、やはり規則性のある数列ではない。

つまり、一部ではあるものの、和音は構成音の周波数比が規則性を持つことで初めて調和を成すことができる。

3. まとめ

周波数をラから1オクターヴ分並べると、周波数は等差数列を成す。
周波数÷11 の比を和音の低い順に並べると、その比は等差数列を成す。
また、それに反比例する弦の長さは調和数列を成す。
不協和音を含むコードは規則性のある数列を成さない。(一部除く)

4. 感想

今までなんとなく曲を聴いていたが、今回関係性を調べてみると、意外なことがいくつも見つかった。曲を聴くときにコードを調べてみて、それがどんな数列を成すのか、あるいは規則性からどの音が構成音に使われているのか確かめてみると面白いかもしれない。逆にまとめに記述したように不協和音の中にも構成音を組み替えることで数列を成すものがあるかもしれないので、それも確かめてみたい。

フェルマーの最終定理の解決に貢献した、日本人数学者

「フェルマーの最終定理」という言葉は多くの人が一度は耳にしたことがあるだろう。この定理は、17世紀の数学者ピエール・ド・フェルマーによって提唱されたもので、次のように述べられている。

x^n + y^n = z^n

この方程式が n > 2 の場合には、正の整数解を持たないというものだ。
つまり、nが2より大きいとき、x, y, z が整数であるような解は存在しないということだ。

この定理は、フェルマーが証明を書き残さなかったため、長い間未解決の問題として残っていた。しかし、1994年にアンドリュー・ワイルズによって証明され、数学界に大きな衝撃を与えた。この証明の話には、3人の日本人数学者の名前がよく登場する。
谷山豊、志村五郎、岩沢健一の3名である。谷村と志村は「谷山―志村予想」、岩沢健一は「岩澤理論」という業績を残し、フェルマーの最終定理の解決に大きく貢献した。彼らの業績について、調べたことをまとめてみる。

谷山―志村予想

谷山–志村予想は、「有理数体上に定義されたすべての楕円曲線はモジュラーである」という予想である。もともとは谷山豊によって提起され、その後志村五郎によって定式化された。簡潔に言うと、楕円曲線（特定の方程式で表される曲線）とモジュラー形式（特定の関数の一種）が深く関係しているということだ。

楕円曲線とは?

実平面上、楕円曲線は次の方程式により定義される平面曲線として表され、次のような法停止kで表される。

y^2 = x^3 + ax + b

ここで、a, bは実数である。

モジュラー形式とは?

複雑な関数の一種で、特定の対称性を持つもの。数論、解析学で重要な役割がある。

岩澤理論

岩澤理論とは、数論の一分野で、特に無限次元のガロア拡大に関する理論。

数の性質を調べるために、数の集合を無限に広げていく方法。例えば、円周上の点を考えると、1周するためには360度(2π[rad])必要だ。この円周上の点を無限に細かく分けていくと、非常に多くの点が得られる。このように、数の集合を無限に広げていくことで、新しい性質や関係性を見つけることができる。

具体例

円分体は1のべき根(例えば、1の2乗根、3乗根、4乗根など)を含む数の集合だ。これを無限に広げていくと、非常に多くの数が含まれることになる。この無限に広がった数の集合を使って、数の性質を調べるのが岩澤理論の一部だ。

ガロア群

ガロア群は、数の集合の対称性を表すものだ。例えば、円周上の点を回転させると、元の位置に戻ることがある。この回転の操作を数学的に表現したものがガロア群だ。

感想

彼らの業績は、いずれも非常に高度な内容のため、さっぱり理解できなかったが、なんとなくの雰囲気を掴むことくらいはできたかもしれない。数学史上屈指の大難問の解決に、日本人が関わっていることは、意外である、(自分が関わっているわけでもないのに)やや誇らしい。私ごときには手も足も出ないが、将来彼らの理論を理解できる丸高生が一人でもいてくれたら嬉しい。

サンクトペテルブルクのパラドックス

偏りのないコインを表が出るまで投げ続け、表が出たときに、賞金をもらえるゲームがあるとする。もらえる賞金は、1回目に表が出たら1円、1回目は表が出て2回目に表が出たら倍の2円、2回目まで表が出ていて3回目に初めて表が出たらそのまた倍の4円、...というふうに倍々で増える賞金がもらえるというゲームをする。
この時の期待値は、

1・1/2 + 2・1/4 + 4・1/8 + … = ∞

となり、無限大に発散する。
よって、賭け金がいくらであっても、この賭けには参加すべきとなる。

しかし、1/2の確率で1円、1/4の確率で2円、1/1024の確率で512円が得られるので、この賭けに1億円かけて儲かるかというと、直感的にはそう思えない。

これに対して、現実的な回答は、賞金には上限があるということだ。例えば、胴元の財産が1億円だとすると、27回続けて裏が出ると、1億円を超えてしまうから、26回裏が出るとゲームを打ち切りにした場合、期待値は14円となる。

よって、現実的な範囲では、期待値はせいぜい数十円となる。

感想

期待値が無限大に発散する場合がある事が面白かった。無限を考慮すると、直感と大きく変わるので、このようなパラドックスが生まれると分かった。このパラドックスには、様々な回答がある、気になったらぜひ調べてみて欲しい。
